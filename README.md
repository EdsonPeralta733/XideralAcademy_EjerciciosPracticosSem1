# Xideral Academy - Ejercicios Prácticos Semana 1

### Edson Miguel Peralta Valdez

Este un proyecto java el cual contiene los ejercicios prácticos realizados en la semana 1 de la Academia Xideral FullStack

## Temas vistos
* Java class structure, packages, imports
* Tipos primitivos vs referencia, operadores, control de flujo
* Strings, access modifiers
* StringBuilder, arrays, ArrayList, wrapper classes
* Constructores, encapsulamiento, herencia
* Clases abstractas, interfaces, polimorfismo
* equals(), hashCode(), toString(), instanceof

## Pasos para correr el proyecto

1. Clonar el repositorio
2. Entrar a cada paquete con su respectivo ejercicio y ejecutar el metodo main
